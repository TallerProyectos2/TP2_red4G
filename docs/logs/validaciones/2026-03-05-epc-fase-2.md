# Validacion de la fase 2 del EPC

- Fecha: `2026-03-05`
- Jira: `TP2-143`
- Maquina: `tp2-EPC`

## Estado inicial

- `srsepc` y `srsenb` se estaban ejecutando manualmente y la configuracion S1 estaba activa
- El EPC no tenia habilitado el enrutamiento de UE:
  - `net.ipv4.ip_forward = 0`
- Faltaba NAT para `172.16.0.0/24`
- El binario `dnsmasq` existia pero no habia ningun servicio `dnsmasq` activo configurado

## Cambio aplicado

- Se hizo copia de seguridad de la configuracion del EPC en `/srv/tp2/config/epc.conf.pre-fase2-20260305`
- Se actualizo `/home/tp2/.config/srsran/epc.conf`:
  - `dns_addr = 172.16.0.1`
- Se habilito el reenvio IPv4 persistente:
  - `/etc/sysctl.d/99-tp2-epc-routing.conf` con `net.ipv4.ip_forward=1`
- Se anadio el bootstrap persistente de enrutamiento UE:
  - script: `/usr/local/sbin/tp2-ue-routing.sh`
  - unidad systemd: `/etc/systemd/system/tp2-ue-routing.service`
  - servicio habilitado e iniciado
- Se instalo y configuro el servicio DNS:
  - paquete `dnsmasq` instalado
  - archivo de configuracion `/etc/dnsmasq.d/tp2-epc.conf`
  - servicio `dnsmasq` iniciado y activo
- Se reinicio `srsepc` una vez para aplicar el parametro DNS del EPC

## Evidencia de ejecucion

- `sysctl -n net.ipv4.ip_forward` -> `1`
- Reglas de `iptables` presentes:
  - `POSTROUTING -s 172.16.0.0/24 -o eno1 -j MASQUERADE`
  - ruta de aceptacion `FORWARD` entre `srs_spgw_sgi` y `eno1`
- `dig +short @172.16.0.1 openai.com` devolvio registros A validos
- `ping -I 172.16.0.1 -c 2 1.1.1.1` tuvo exito con `0%` de perdida de paquetes
- El contador de impactos NAT aumento en la regla MASQUERADE de origen UE
- Se observo una nueva configuracion S1 tras el reinicio del EPC en la consola de `srsepc`
- El lado eNodeB confirma S1 `ESTAB` hacia el EPC (`10.10.10.1:36412`)

## Comprobacion pendiente

No se observaron nuevos eventos de attach de UE ni de plano de datos durante la corta ventana de observacion posterior al cambio. El criterio final que requiere confirmacion explicita de ping/HTTP desde el lado UE sigue pendiente hasta que el UE genere trafico a traves de la ruta del EPC.
